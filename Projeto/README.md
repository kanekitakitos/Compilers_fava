## Shared Architecture

Across the three stages, the implementation is organized around the same core layers:

- `Antlr4` for grammar definition
- `Compiler` for orchestration and error handling
- `Semantic` for type rules and symbol-table logic
- `CodeGenerator` for bytecode emission
- `VirtualMachine` for program execution

## Supported Flags

| Flag | Meaning |
|---|---|
| `-asm` | prints generated bytecode or assembly-like output |
| `-trace` | traces VM execution |
| `-showLexerErrors` | shows lexer diagnostics |
| `-showParserErrors` | shows parser diagnostics |

## Example Usage

```bash
java FavaCode.FavaCompileAndRun input/T1input.fava -asm -trace
```
